üìã Contribution Guidelines

1. Fork the Repository

Start by forking our GitHub repository to your own account.

2. Create a Branch

Create a descriptive branch name for your contribution (e.g., add-thermodynamics-examples or fix-mobile-navigation).

3. Make Your Changes

Follow our coding standards and content guidelines when making changes:

  • Use clear, concise language for physics explanations
  • Include proper mathematical notation using LaTeX/MathJax
  • Test your changes locally using Jekyll
  • Ensure mobile responsiveness

4. Submit a Pull Request

When ready, submit a pull request with:

  • Clear description of changes made
  • Screenshots for visual changes
  • Testing information
  • Reference to any related issues
