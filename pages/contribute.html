Getting Started

  1. Visit our GitHub repository: PhysicsDaily.github.io
  2. Fork the repository to your GitHub account
  3. Make your changes in your fork
  4. Submit a pull request with a clear description of your changes

Contribution Guidelines

  • Ensure all physics content is accurate and well-sourced
  • Follow the existing code style and formatting
  • Test your changes before submitting
  • Write clear commit messages
  • Be respectful in all interactions

Questions?

If you have any questions about contributing, please:

  • Open an issue on GitHub for discussion
  • Check our existing issues for similar questions
  • Contact us through our About page
