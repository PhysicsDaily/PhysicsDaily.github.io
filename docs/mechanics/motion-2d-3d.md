## Preview: Key Equations

$$\vec{r} = x\hat{i} + y\hat{j} + z\hat{k}$$

$$\vec{v} = \frac{d\vec{r}}{dt}$$

$$\vec{a} = \frac{d\vec{v}}{dt}$$

**Projectile Motion:**

$$x = v_0 \cos\theta \cdot t$$

$$y = v_0 \sin\theta \cdot t - \frac{1}{2}gt^2$$
