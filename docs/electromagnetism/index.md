## Key Concepts Preview

!!! info "Fundamental Equations"
    **Coulomb's Law:**
    $$F = k\frac{q_1 q_2}{r^2}$$
    
    **Gauss's Law:**
    $$\oint \vec{E} \cdot d\vec{A} = \frac{Q_{enc}}{\epsilon_0}$$
    
    **Faraday's Law:**
    $$\mathcal{E} = -\frac{d\Phi_B}{dt}$$
    
    **Maxwell's Equations** unify all of electromagnetism!
