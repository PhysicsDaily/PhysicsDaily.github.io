## Key Concepts Preview

!!! info "Wave Fundamentals"
    **Wave Equation:**
    $$y(x,t) = A \sin(kx - \omega t + \phi)$$
    
    **Wave Speed:**
    $$v = f\lambda = \frac{\omega}{k}$$
    
    **Sound Intensity Level:**
    $$\beta = 10 \log_{10}\left(\frac{I}{I_0}\right) \text{ dB}$$
